What’s wrong with vagrantbox.es?

Nothing! I’m a big fan of that project. However, they seem to be going through some scaling issues with the number of pull requests that they receive. They’re a little slow at integrating them back in and I wonder if it’s because everyone is editing a single HTML file.

This approach aims to separate more of the data from the presentation by “creatively” using the Jekyll post system.

Hopefully upstream will like the approach and integrate it back in. :)
